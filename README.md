# SemanticTruth AI – FactGuard Intelligence

SemanticTruth AI – FactGuard Intelligence is an AI-powered fake news detection system that uses Natural Language Processing (NLP), semantic analysis, and machine learning to verify the authenticity of news articles. It analyzes content patterns, writing style, and source credibility to provide accurate, reliable, and real-time misinformation detection.


# Features

- Detect fake and real news in text or from URLs  
- Real-time confidence scoring for predictions  
- Uses TF-IDF vectorization and Multinomial Naive Bayes for classification  
- User-friendly web interface built with flask
- Scrapes content from online news articles using BeautifulSoup

# Demo

Enter a news URL or paste text in the app to check if it is real or fake. The system will display:

- Confidence score (%)
